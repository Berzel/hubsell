## How to run locally

0. Make sure you've nodejs and git installed on your computer
1. Clone this repository to your computer
2. `cd` into the project folder
3. Run `npm install`
4. Run `npm run dev`
5. Visit [localhost:5173](http://localhost:5173/) on your local machine
